# Coffe Lawhore

## ☕ Descripción

Coffe Lawhore es una página web de presentación para una cafetería con un estilo elegante y moderno.

## 🎨 Diseño inspirado en café

El sitio usa una paleta cálida de café, crema y verdes suaves, y presenta un logo en enlace de imagen con estilo renovado. Está pensado para transmitir la experiencia del café artesanal: elegante, acogedora y visualmente atractiva.

## 📁 Estructura del proyecto

- `index.html` — página principal con la estructura del sitio.
- `CSS/style.css` — estilos y diseño visual del sitio.
- `js/main.js` — script principal para interacciones y animaciones.
- `img/` — carpeta donde debe ir la imagen del logo y otros recursos gráficos.

## ✅ Funcionalidades principales

- Navegación fija y scroll suave.
- Carrusel de hero con varias bebidas destacadas.
- Sección de bebidas y descripciones.
- Sección "Sobre nosotros" con diseño enfocado en relato de marca.
- Formulario de suscripción para captar correos.
- Pie de página con enlaces y redes sociales.
